#### 원래 있었던 자리에서 위치이동
- position: static;
- position: relative;

#### 가장 가까운 박스 안에서 위치이동
- position: absolute;

#### 페이지 상에서 위치이동
- position: fixed;

#### 스크롤 시 붙음
- position: sticky;

#### flex시 할 수 있는 것
- display: flex;
- flex-direction: row;
- flex-wrap: warp;
- flex-flow: column nowarp;
